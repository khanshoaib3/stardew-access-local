## Changelog v1.6.2

### Feature Updates

- Updated inventory page's money key to only speak festival/event items during the festival/event and also included date string to the info.
- Using the `HealthNStamina` key in the inventory page will now speak the active buffs and debuffs along with the health and stamina.

### Bug Fixes

- Fixed GM not triggering end of festival dialogue.
- Added previously completed status on special orders board.
- Properly stop pathfinding when player isn't free to move and also stop animating.
- Fixed pet adoption menu.
- Fixed boots not speaking defense and immunity bonuses properly.
- Fixed water troughs in greenhouse and slime hutch and also made them dynamic in slime hutch like feeding benches.

### Tile Tracker Changes

- Added desert festival, squid fest and trout derby related tiles.
- Tracked shrine of challenge and skull statue (in skull cave).

### Misc

- Added gender to slimes.
- Speak calico statue effect string and race announcer's texts.
